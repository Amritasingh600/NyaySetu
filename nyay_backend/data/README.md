# NyaySetu Data Directory

Place deployment artifacts that the backend needs at runtime in this folder. Typical contents:

- `bns.pdf` — the authoritative Bharatiya Nyaya Sanhita corpus used by the LegalVectorStore.
- `chroma_db/` — persistent Chroma embeddings generated by `NyaySetuIntelligence.build_index()`.

These files are not committed to version control. Use environment variables (`NYAYSETU_BNS_PDF_PATH`, `NYAYSETU_CHROMA_DIR`) to override their locations in production.
